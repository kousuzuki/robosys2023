# robosys2023

標準入力から読み込んだ数字を足す

### 11/29 機能追加
*合計することに加えて、入力した数値の数、平均を表示可能。

## インストール方法
*お使いの環境で以下を入力してください。
```bash
git@github.com:kousuzuki/robosys2023.git
```

## 使用方法について
```bash
* cd robosys2023

* chmod +x plus

* seq 5 | ./plus 
```

以下が出力した結果です

```bash
$ seq 5 | ./plus

15.0 5 3.0
```



## 必要なソフトウェア
* Python

## テスト環境
* Ubuntu 20.04 on Windows
